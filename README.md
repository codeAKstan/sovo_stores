## Maintenance mode

- Set `MAINTENANCE_MODE` in your environment to control access.
- When enabled, requests are redirected to `/maintenance` except for `/_next`, `/favicon*`, `/maintenance`, `/api/*`, `/admin/*`.
- Example:

```
MAINTENANCE_MODE=true
```
